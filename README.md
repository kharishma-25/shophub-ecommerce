## Tech stack and Deliverables : 

Web Admin Portal (Next.js + TypeScript) - For admins to manage products, inventory, orders, and view analytics
Mobile Shopping App (iOS SwiftUI + Android Compose) - For customers to browse products, add to cart, checkout, and track orders
Backend Microservice (Java Spring Boot) - RESTful API with PostgreSQL database for all e-commerce operations
Database Design - Comprehensive schema for products, orders, cart, users, categories
Confluence Documentation - Complete technical documentation
GitHub Repository - Production-ready code with CI/CD setup
